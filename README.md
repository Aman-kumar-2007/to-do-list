📌 To-Do List App

A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript. This app helps users manage daily tasks efficiently by allowing them to add, delete, and mark tasks as completed.

🚀 Features
➕ Add new tasks
✅ Mark tasks as completed
❌ Delete tasks
💾 Data stored using Local Storage (no backend required)
🛠️ Tech Stack

HTML – Structure of the app
CSS – Styling and layout
JavaScript – Functionality and logic
Local Storage – To save tasks in the browser
